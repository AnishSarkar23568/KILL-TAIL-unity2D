# KILL-TAIL-unity2D

 built in **Unity 2D using C#**.  
This project demonstrates grid-based movement, snake body growth, collision detection, scoring system, and game-over mechanics.

---

# 🎮 Gameplay

The player controls a snake that moves across a grid.

The snake grows when eating humen and the game ends when it collides with:
- Walls
- Its own body

---

# ✨ Features

- Grid-based snake movement
- Snake body growth system
- Movement history system for smooth body following
- Humen  collection mechanics
- Score system
- Game Over system
- Camera shake effects
- Sound effects and background music
- Blood particle effects when eating food

---

# 🧠 Core Systems Implemented

### Snake Movement
The snake moves in a grid-based system using a movement timer.

### Body Follow System
The snake body follows the head using a **movement history list**, storing previous positions and directions.

### Growth Mechanic
Each time food is collected, a new body segment is instantiated using a prefab.

### Collision System
The snake detects collisions with:
- Walls
- Food objects
- Its own body

### Game Over System
When the snake collides with a wall or its body, the game triggers:
- Camera shake
- Sound effects
- Game Over UI
- Time freeze
